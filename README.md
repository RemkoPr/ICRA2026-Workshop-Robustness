# ICRA2026-Workshop-Robustness

This repository is set up so a PDF is readable directly on GitHub Pages.

## How to publish your PDF

1. Replace `/workshop.pdf` with your own PDF file (keep the same name).
2. Enable GitHub Pages for this branch/repository (root folder).
3. Open the Pages URL (for example: `https://<user>.github.io/ICRA2026-Workshop-Robustness/`).

The `index.html` page embeds `workshop.pdf` in an in-browser viewer.
